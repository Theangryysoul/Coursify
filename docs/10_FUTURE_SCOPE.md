# 1. Purpose

This document lists features intentionally excluded from Version 1.

These ideas should only be considered after the current version is stable.

The goal is to prevent feature creep while providing a clear roadmap for future development.

---

# 2. Vision

Coursify aims to become the best personal learning companion for online educational content.

The focus will always remain:

- Learning
- Consistency
- Organization
- Progress

Rather than entertainment or social engagement.

# 9. Product Philosophy

Every future feature must answer one question:

> "Does this help users learn more effectively and consistently?"

If the answer is **No**, the feature should not be added.

Growth should never come at the cost of simplicity.

The product should remain:

- Focused
- Fast
- Distraction-free
- Easy to use
- Easy to maintain

---

# Future Vision

Coursify is not intended to replace learning platforms.

Its purpose is to organize, simplify, and improve the learning experience across platforms.

The long-term goal is to build a personal learning companion that helps users stay consistent, understand their progress, and achieve their learning goals.
